
# 占い師の部屋への入室体験アニメーション

## 概要
「占いを始める」ボタンを押した後、占い師の部屋に入っていくような没入感のあるトランジションアニメーションを実装します。

## 現在の動作
1. ユーザーが占い師を選択
2. 「占いを始める」ボタンをクリック
3. 即座に接続処理が開始され、`ConnectedAgentDisplay`が表示される

## 提案する新しい動作
1. ユーザーが占い師を選択
2. 「占いを始める」ボタンをクリック
3. **[新規] 入室アニメーションが開始**
   - 画面がフェードアウト（暗転）
   - 神秘的なパーティクルエフェクト
   - 占い師の画像が大きくズームイン
   - 「○○の部屋に入ります...」というテキスト表示
4. アニメーション完了後、通常の接続処理が開始
5. 接続完了後、占いセッション開始

## 実装内容

### 1. EnterRoomTransition コンポーネント（新規作成）

フルスクリーンのオーバーレイで入室アニメーションを表示：

```text
+------------------------------------------+
|                                          |
|     ✦ ✧ ✦ (神秘的なパーティクル)        |
|                                          |
|        ┌─────────────────┐               |
|        │                 │               |
|        │  占い師の画像    │  ← 大きく表示  |
|        │  (ズームイン)    │               |
|        │                 │               |
|        └─────────────────┘               |
|                                          |
|     「神秘の間へようこそ...」             |
|                                          |
+------------------------------------------+
```

**アニメーションシーケンス：**
- 0-0.3秒: 画面全体がフェードイン（暗転）
- 0.3-1.0秒: 占い師の画像が中央からズームイン
- 1.0-1.5秒: 神秘的な光のエフェクト
- 1.5-2.5秒: ウェルカムメッセージ表示
- 2.5-3.0秒: フェードアウトして本来のUI表示

### 2. VoiceChat / TextChat の修正

入室アニメーション状態を管理：
- `showEnterAnimation: boolean` - アニメーション表示中かどうか
- ボタンクリック時にアニメーションを開始
- アニメーション完了後に実際の接続処理を開始

### 3. CSS アニメーション追加

新しいキーフレームアニメーション：
- `zoom-in-fade`: 画像のズームイン
- `glow-pulse`: 神秘的な光のパルス
- `particle-float`: パーティクルの浮遊
- `text-reveal`: テキストの登場

### 4. 占い師ごとのカスタマイズ

各占い師の個性に合わせた演出：
- 占い師の名前と画像を使用
- 背景色やエフェクトを占い師のテーマカラーに合わせる
- ウェルカムメッセージをパーソナライズ

## 技術詳細

### EnterRoomTransition Props
```typescript
interface EnterRoomTransitionProps {
  agent: Agent;          // 選択された占い師
  isVisible: boolean;    // 表示/非表示
  onComplete: () => void; // アニメーション完了時のコールバック
  displayName?: string;  // ユーザーの表示名
}
```

### アニメーションの特徴
- 全画面オーバーレイで没入感を演出
- 占い師の画像を大きく表示（画面の60-70%を占める）
- 神秘的なパーティクルエフェクト（星や光の粒子）
- グラデーション背景で深みのある空間を表現
- 3秒程度の適切な長さ（長すぎず短すぎず）

## ユーザー体験の向上ポイント

1. **没入感**: フルスクリーンアニメーションで日常から占いの世界へ引き込む
2. **期待感**: 占い師の大きな画像で「これから占ってもらう」という実感
3. **神秘性**: 光のエフェクトやパーティクルで魔法のような雰囲気
4. **パーソナライズ**: ユーザー名での呼びかけで特別感を演出
